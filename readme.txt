Professional Arabic typography plugin using the official LPMQ Isep Misbah font. Supports multi-line Surah and clean RTL layout.

== Description ==

**(ng)Arab** (pronounced 'Ngarab') is designed for bloggers, scholars, and developers who want to display Arabic text, such as Quranic verses or Hadiths, with professional-grade typography.

The plugin utilizes the **LPMQ Isep Misbah** font, which is the standard for Islamic documentation in Indonesia, ensuring maximum legibility and aesthetic appeal.

= Key Features =
* **Standardized Font**: Implemented using the lightweight WOFF2 format for fast loading times.
* **Multi-line Support**: Perfectly handles long Surahs and multi-paragraph Arabic text without breaking the layout.
* **Theme Compatible**: Uses forced CSS styles to ensure your Arabic text looks perfect regardless of the WordPress theme you are using.
* **Responsive Design**: Adapts beautifully to mobile devices and tablets.

== Installation ==

1. Upload the `arabic` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[ngarab]` shortcode to wrap your Arabic text, like this: `[ngarab] Arabic text here [/ngarab]`.

== Frequently Asked Questions ==

= How do I use the shortcode? =
Simply wrap any Arabic text with `[ngarab] ... [/ngarab]`. It works for both short phrases and long paragraphs.

= Does it support RTL (Right-to-Left)? =
Yes, the plugin automatically handles RTL direction and right-text alignment for the Arabic blocks.
